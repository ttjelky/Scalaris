# Scalaris — Frontend

React + Vite frontend for the Scalaris mobile-first activity app.

## Development

```bash
npm install
npm run dev         # https://localhost:5174
```

## Tests

```bash
npm run test
npm run test:watch
```

## Build

```bash
npm run build
npm run preview
```

See the [root README](../README.md) for full setup instructions.
