# Scalaris — Frontend

React 19 + Vite 8 + Leaflet (map) + React Router 7

This is the frontend for the Scalaris social map app. It runs locally via `npm run dev` on `https://localhost:5174`.

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start Vite dev server (HTTPS, hot-reload) |
| `npm run build` | Production build → `dist/` |
| `npm run test` | Run Vitest unit tests (single run) |
| `npm run test:watch` | Run Vitest in watch mode |
| `npm run lint` | ESLint check |
